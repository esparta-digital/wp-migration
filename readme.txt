== Description ==
This plugin is a modified version of the All-in-One WP Migration plugin by ServMask Inc.
Modifications by Esparta Digital for use with agency clients:
- Custom update system
- Disabled automatic WordPress.org plugin updates
- Other internal tweaks for client projects

Original plugin URI: https://es.wordpress.org/plugins/all-in-one-wp-migration/
Original Author: ServMask Inc.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/wp-migration/` directory, or install via GitHub for internal use.
2. Activate the plugin through the 'Plugins' screen in WordPress.
